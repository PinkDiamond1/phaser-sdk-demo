# PHASER SDK DEMO

A basic example showcasing a simple HTML5 game interacting with a Loom DappChain, using [Phaser](http://phaser.io) and  [Loom.js](https://github.com/loomnetwork/loom-js).


Run dev server

```
# NPM
yarn run dev
# Yarn
npm run dev
```

Protobuf compilation

```
# NPM
yarn run proto
# Yarn
npm run proto
```
